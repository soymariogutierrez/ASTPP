
==================== 
Local Number
====================

.. image:: /Images/local_number_logo.png
	:height: 100px
  

**Purpose**
 - Routing one live number to the multiple end customers.

 
**Stpes for install**
 - Click on the **Get Addons**.
 - Click on the **Community**.
 - Click on the **install** button from **Lcoal Numbers**.

**Menu Location**

 Inbound -> Local Numbers
 
 .. image:: /Images/local_number_list.png
	
 
**Field Level Description** 

   
========  	==================================		================================================================== 
SR NO  		Value  	   								Description  
========  	==================================		================================================================== 
1      		Number    								Please enter the live local number

2			Country									Please select the country where the number belongs 

3			Province / State						Please enter the Province / State based on the country selection.

4			City									Please enter the city based on the country selection.

5			Status									Please select the status.
