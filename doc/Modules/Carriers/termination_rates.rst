=================
Termination Rates
=================

Termination rates belongs to Trunk. Termination rates works as carrier rates / buy rates as these rates will be applicable on providers.
Before dialing number to gateway, system will do LCR / COST based routing process on termination rates and find out best matched code and dial out using that.
