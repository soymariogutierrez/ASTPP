==================
Company Profiles
==================

**Purpose**

 - Reseller Personalization it means create customer under the reseller portal.
 - During create reseller we have also create that reseller company profile.
 - If reseller purchase our portal and set that company profile like logo, domain, header, footer etc.
 - Then after using reseller domain under signup customer it’s all reseller customer.
 - All the company profile related impact display on portal it’s called reseller personalization.
