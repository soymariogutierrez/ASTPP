===================
Create IP map(ACL)
===================


.. image:: /Images/Create_IP_map_3.0.png
  
  
  
**IP Map Field Description**
  
===========  ==================================================================  
**Account**	 Select account number you wish to assign IP based authentication.
  
**Name**	   Give name of IP map.
  
**IP**	     Enter IP address.
  
**Prefix**	 Enter prefix value.
===========  ==================================================================
